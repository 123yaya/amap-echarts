amap-echarts

在高德地图上展示 echarts 图表

快速上手 →

灵活

amap 对象由外部传入，方便与 vue-amap 或者其他框架的 amap 组件配合使用。

轻量

类库只提供 echarts 坐标转换与刷新时机，使用方式不变，可按需加载。

一致

除了 Promise 化，实例接口与 echarts 保持一致，减轻 api 负担。
